# Chaturbate CLI

cb-cli is a bash script to watch chaturbate streams in mpv from the terminal. You can subscribe to your favorite streamers, watch or download from a live stream, or print a help menu. That's it for now.

# Installation

Dependencies:

    grep
    curl
    mpv
    yt-dlp

To install, simply run the following command;

    sudo curl -L -o /usr/bin/cb-cli https://raw.githubusercontent.com/tapebysea/cb-cli/refs/heads/master/main/cb-cli && sudo chmod +x /usr/bin/cb-cli

Watch and download chaturbate streams from the terminal
